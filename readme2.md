# epochs
dcgan cat training from scrath
input:
    - afhq cat dataset (256x256)
output:
    - 64x64 cat generated images  during training
    - `generator_final.pth`
        model generated from dcgan training

# cat_inference
inference using dcgan `generator_final.pth` model
input:
    - noise
output:
    - 64x64 cat generated images at `\cats_inference`

# cats_upscaled
upscaling using real-esrgan on cat_inference
input:
    64x64 images generated by dcgan `\cats_upscaled`

# epochs_finetune
finetuning of basemodel srgan
input:
    - afhq cat dataset (256x256)
output:
    - 256x256 cat generated images during finetuning
    - `sr_generator_finetuned_final.pth`
        model generated from srgan finetuning

# cats_upscaled_finetuned
input:
    64x64 images generated by dcgan at `\cats_inference` (64x64)
output
    256x256 images upscaled by srgan at `\cats_upscaled_finetuned` (64x64)
